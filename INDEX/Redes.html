Redes de Computadores

São sistemas interconectados de computadores e dispositivos que permitem a comunicação e o compartilhamento de recursos, como arquivos e impressoras. Essas redes podem ser locais (LANs), abrangendo um espaço físico limitado, como uma residência ou escritório, ou podem ser redes mais amplas, como a internet.

Tipos de redes
  • PAN - Personal Area Network
  • CAN - Controller Area Network
  • LAN - Local Area Network
  • MAN - Metropolitan Area Network
  • WAN - Wide Area Network

Topologia

É um termo utilizado para designar a configuração assumida pelos computadores de Rede, ou seja, a forma como estes estão interligados.

  1. Topologia Lógica: refere-se ao modo como os dados circulam entre os computadores da rede
  2. Topologia Física: refere-se à forma como os computadores e cabos de rede se interligam fisicamente na Rede.
Tipos de Topologia Física
  • - Barramento
  • - Anel
  • - Estrela
  • - Duplo anel
  • - Estrela estendida
  • - Arvore ou Hierárquica
  • - Malha
  • - Mista

    Modelo OSI - Open Systems Interconnection

    É um modelo de referência que descreve como os sistemas de computadores podem se comunicar uns com os outros. Ele foi desenvolvido pela International Organization for Standardization (ISO) e é amplamente utilizado como base para o projeto e implementação de redes de computadores.

    Estrutura do modelo OSI
    1. Física
    2. Enlace → MAC
    3. Rede → IPV4 / IPV6
    4. Transporte → (UDP / TCP) Forma de transporte
    5. Seção
    6. Apresentação
    7. Aplicação → HTTP/TTTPS/FTP/FTPS

    Modelo TCP/IP

    O modelo TCP/IP é um conjunto de protocolos de comunicação utilizado em redes de computadores para a troca de informações. Ele foi desenvolvido no final da década de 1970 e é amplamente utilizado como base para a internet.

    Estrutura do modelo TCP/IP
    1. Acesso a rede→ MAC
    2. Internet→ IPV4 / IPV6
    3. Transporte→ UDP / TCP
    4. Aplicação HTTP/TTTPS/FTP/FTPS

    Camada 3 - Rede

    IPv4 - Internet Protocol version 4

    É uma versão do protocolo de Internet amplamente utilizado para a comunicação em redes de computadores. Foi a primeira versão do protocolo a ser amplamente implantada e é a base da atual infraestrutura da Internet.

    Ele utiliza endereços de 32 bits, representados por quatro grupos de números de 0 a 255, separados por pontos.

  • Por exemplo, um endereço IPv4 pode ser 192.168.0.1.

    IPv6 - Internet Protocol version 6

    É a versão mais recente do protocolo de Internet utilizado para identificar e localizar dispositivos na rede. Ele foi desenvolvido para substituir o IPv4, que está se esgotando devido ao crescimento contínuo da Internet e ao aumento do número de dispositivos conectados.

    O IPv6 usa endereços de 128 bits, o que permite um número virtualmente ilimitado de endereços disponíveis.

  • Um exemplo de endereço IPv6 pode ser 2001 :db8:3333:4444:5555:6666:1.2.3.4
