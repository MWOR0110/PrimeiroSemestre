Amostragem

Amostragem é o processo de seleção de uma parte representativa de uma população maior para realizar análises e tirar conclusões sobre o todo. Em vez de examinar todos os elementos da população, uma amostra é selecionada e os resultados obtidos dessa amostra são extrapolados para a população maior. A amostragem é usada em diversas áreas, como pesquisas de mercado, pesquisas científicas e estatísticas, fornecendo uma abordagem mais eficiente e prática para obter informações sobre uma população.

  • Aleatória:

    • é um método de seleção de amostras em que todos os elementos da população têm uma chance igual de serem escolhidos. É um processo imparcial e objetivo.

  • Sistemática:

    • Um método de seleção de amostras em que os elementos da população são escolhidos de acordo com um padrão sistemático. Isso envolve selecionar aleatoriamente o primeiro elemento da população e, em seguida, escolher os demais elementos com base em um intervalo fixo, pulando uma determinada quantidade de elementos.

  • Estratificada:

    • Seleção de amostras em que a população é dividida em subgrupos homogêneos chamados estratos. Cada estrato é composto por elementos que possuem características semelhantes dentro do grupo. Em seguida, uma amostra é selecionada aleatoriamente de cada estrato de forma proporcional ao tamanho do estrato em relação à população total.

Tendência Central

Tendência central refere-se à medida estatística que representa o valor central ou típico de um conjunto de dados. É usada para resumir e descrever a localização central dos dados.

  • Média

    • É a soma de todos os valores em um conjunto de dados dividida pelo número de observações. A média é sensível a valores extremos e é amplamente utilizada quando os dados estão distribuídos de forma aproximadamente simétrica.

  • Mediana

    • É o valor central que divide o conjunto de dados em duas partes iguais, onde metade dos valores está abaixo e metade está acima dela. A mediana é menos afetada por valores extremos e é adequada para dados com distribuição assimétrica.

  • Moda

    • É o valor que ocorre com maior frequência em um conjunto de dados. Pode haver uma ou mais modas, ou pode não haver moda se todos os valores ocorrerem com a mesma frequência. A moda é útil para dados categóricos ou discretos.

Medidas de Dispersão

são estatísticas que fornecem informações sobre o grau de variação ou dispersão dos dados em um conjunto de observações. Elas mostram o quão afastados ou próximos os valores estão em relação a uma medida de tendência central, como a média.

  • Amplitude:

    • É a diferença entre o maior e o menor valor do conjunto de dados. Ela fornece uma ideia geral da variação total, mas pode ser influenciada por valores extremos.

  • Desvio padrão:

    • É uma medida que indica o quanto os valores do conjunto de dados se desviam, em média, em relação à média. O desvio padrão é amplamente utilizado e fornece uma medida de dispersão mais precisa. Valores maiores indicam maior dispersão dos dados.

  • Variância:

    • É o quadrado do desvio padrão e representa a média dos quadrados das diferenças entre cada valor e a média. A variância também é utilizada para medir a dispersão dos dados, sendo uma medida comumente empregada em cálculos estatísticos.

  • Intervalo interquartil (IQR):

    • É a diferença entre o terceiro quartil (Q3) e o primeiro quartil (Q1). O IQR é útil para medir a dispersão dos dados enquanto minimiza a influência de valores extremos.

Teste de Hipóteses

O teste de hipóteses é uma técnica estatística usada para tomar decisões ou tirar conclusões sobre uma afirmação ou hipótese relacionada a uma população com base em uma amostra de dados. Ele permite avaliar se uma determinada afirmação é estatisticamente suportada pelos dados disponíveis ou se deve ser rejeitada em favor de uma hipótese alternativa.

  • Qui Quadrado

    • é um teste estatístico utilizado para avaliar a associação entre duas variáveis categóricas. Ele compara as frequências observadas em uma tabela de contingência com as frequências esperadas, assumindo que não há associação entre as variáveis. O teste determina se existe uma diferença significativa entre as frequências observadas e as frequências esperadas, o que indicaria uma associação entre as variáveis.

  • Kolmogorov-Smirnov

    • Ele compara a distribuição empírica dos dados (observados) com a distribuição teórica especificada (esperada) e calcula uma estatística de teste que representa a diferença entre as duas distribuições.

  • Anderson Darling

    • É uma extensão do teste de Kolmogorov-Smirnov, que enfatiza as diferenças nas caudas da distribuição.

Inferência

uma área da estatística que se concentra em tirar conclusões sobre uma população com base em informações limitadas de uma amostra. Ela envolve o uso de técnicas estatísticas para fazer estimativas, testar hipóteses e tomar decisões sobre uma população com base em dados amostrais.

  • Distribuição probabilística binomial

    • A distribuição binomial é um modelo probabilístico utilizado para descrever a ocorrência de um evento binário em um determinado número de ensaios independentes e idênticos. Ela é aplicada quando há apenas dois resultados possíveis para cada ensaio, geralmente chamados de "sucesso" e "fracasso".

    P(X = k) = C(n, k) * p^k * (1 - p)^(n - k)

  • Regressão logistica

    • É um modelo estatístico utilizado para realizar a análise de variáveis independentes (preditoras) em relação a uma variável dependente binária (ou dicotômica). É uma técnica de modelagem estatística que estima a probabilidade de ocorrência de um evento ou resultado específico.

Derivadas

As derivadas são um conceito fundamental da matemática que descreve a taxa de variação de uma função em relação a uma variável independente. Elas nos permitem determinar a inclinação da curva em um determinado ponto e entender como a função está se comportando localmente.

A derivada de uma função f(x) em relação à variável x é denotada por f'(x) ou dy/dx. Matematicamente, a derivada é definida como o limite do quociente de diferenças entre os valores da função em dois pontos próximos, à medida que esses pontos se aproximam infinitamente.
