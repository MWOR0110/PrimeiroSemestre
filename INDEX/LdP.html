HTML - Hyper Text Markup Language

É a linguagem de marcação padrão utilizada para a criação e estruturação de páginas da web. É a base fundamental para a construção de sites e aplicativos web. O HTML consiste em um conjunto de elementos ou tags que são utilizados para descrever a estrutura e o conteúdo de uma página web.

HTML semâmtico

Refere-se ao uso adequado de elementos HTML para atribuir significado e estrutura a um documento web.

CSS - Cascading Style Sheets

É uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML.

  • Criação de animações
  • Criação de efeitos visuais
  • Sites dinâmicos
  • Landings pages

Git Hub

Uma plataforma com um banco de armazenamento com códigos de programação, além de auxiliar os programadores em seus projetos, também é uma rede de compartilhamento e troca de conhecimento entre os profissionais dessa área.Ele é usado para controlar o histórico de alterações de arquivos e principalmente de projetos de desenvolvimento de software.

Alguns comandos simples

  • "git init" - Inicializa o repositório
  • "git status" - Verifica se houve alterações / estado dos arquivos
  • "git clone" - Clonar Repositório

JavaScript/JS

É uma linguagem de programação interpretada de alto nível, o JavaScript é a linguagem padrão que os navegadores interpretam e que com HTML e CSS formam a base de toda a Web.

Operadores

Em JavaScript, os operadores são símbolos especiais utilizados para realizar operações em valores ou variáveis. Existem vários tipos de operadores em JavaScript, que podem ser classificados em diferentes categorias. A seguir, estão alguns dos operadores mais comuns, clique para ver a simbologia:

Operadores de atribuição:

Operadores de atribuição são utilizados em programação para atribuir valores a variáveis.

  • ( = ) - Atribui um valor a uma variável.
  • ( += ), ( -= ), ( *= ) e ( /= ) - Realiza uma operação e atribui o resultado a uma variável.

Operadores aritméticos:

Os operadores aritméticos são símbolos usados para realizar operações matemáticas básicas em valores numéricos.

  • ( + ) - Soma dois valores ou concatena strings.
  • ( - ) - Subtrai dois valores.
  • ( * ) - Multiplica dois valores.
  • ( / ) - Divide dois valores.
  • ( % ) - Retorna o resto da divisão.
  • ( ++ ) - Incrementa um valor em um.
  • ( -- ) - Decrementa um valor em um.

Operadores de comparação:

São utilizados para comparar valores e retornar um resultado verdadeiro ou falso, dependendo da relação entre esses valores.

  • ( == ) - Verifica a igualdade de valores.
  • ( === ) - Verifica a igualdade de valores e tipos de dados..
  • ( != ) - Verifica a desigualdade de valores.
  • ( !== ) - Verifica a desigualdade de valores ou tipos de dados.
  • ( < ), ( >), ( <= ), ( >= ) - Verifica as relações de ordem entre dois valores.

Operadores lógicos:

Os operadores lógicos são utilizados para realizar operações de lógica booleana em expressões e são amplamente utilizados na programação.

  • ( && ) - Retorna true se ambos os operandos forem verdadeiros.
  • ( || ) - Retorna true se pelo menos um dos operandos for verdadeiro.
  • ( ! ) - Nega um valor booleano.

Operadores ternários:

Operadores ternários, também conhecidos como operadores condicionais, são utilizados em programação para avaliar uma condição e retornar um valor com base nessa avaliação.

  • ( condição ? expr1 : expr2 ) - Avalia uma condição e retorna expr1 se for verdadeira, caso contrário, retorna expr2.
